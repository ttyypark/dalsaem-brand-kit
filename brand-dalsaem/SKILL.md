# 달샘 브랜드 스킬

> "내 브랜드로 ○○ 만들어줘"라고 하면 이 파일의 색·폰트·톤이 자동 적용된다.

## 브랜드 정체성 요약

- **브랜드명:** 달샘
- **슬로건:** 조용한 샘에 비추는 달처럼
- **보조 카피:** 혹은 달 품은 샘처럼 복잡한 머릿속을 고요하게 정리해드립니다
- **카테고리:** 개인 지식 정리 / 감성 아카이빙
- **레퍼런스:** Ness Labs (nesslabs.com)

## 말투 · 톤

- **조용하고 따뜻하게.** 강요하지 않고 옆에서 같이 앉아주는 느낌.
- 청중: 정보는 많은데 체계가 없어 답답한 사람. 차갑게 관리하기보다 감성 있게 정리하고 싶은 사람.
- 말하지 않는 것: 빠름, 효율, 생산성. 대신 고요함, 정돈, 나다움.

## 디자인 토큰 (항상 이 값 고정 사용)

| 역할 | HEX |
|------|-----|
| 주색 (딥 인디고) | `#2D3A5F` |
| 배경 (크림 화이트) | `#F9F7F2` |
| 표면 (따뜻한 페이퍼) | `#F0EDE5` |
| 본문 (깊은 먹색) | `#1C1C1C` |
| 보조 텍스트 | `#6E6B65` |
| 포인트 (달빛 골드) | `#B8924A` |
| 라인 | `#DDD9D0` |

- **헤딩 폰트:** Noto Serif KR (부드러운 명조)
- **본문 폰트:** Pretendard
- **바탕:** 밝은 크림 톤 (#F9F7F2)

## 산출물 적용 시 체크리스트

- [ ] 헤딩은 항상 Noto Serif KR + `#2D3A5F`
- [ ] 배경은 `#F9F7F2` (어두운 배경 금지)
- [ ] 포인트색은 `#B8924A` 하나만 (골드 계열 혼용 금지)
- [ ] 말투: 차분하고 조용하게 — 느낌표·대문자 과용 금지
- [ ] 워드마크는 `mark.svg` 그대로 사용
